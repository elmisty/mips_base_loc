• 내부 IP 설정
    • 기존에 설정된 내부 네트워크 설정이 기본 공유기의 설정 (192.168.0.xxx)과 다른경우에 사용합니다.
    • 예를 들어, 회계관리서버의 IP 주소가 192.168.10.10으로 고정되어 있는 경우에 내부 IP 주소는 192.168.10.1로 변경하여 사용하시면 됩니다.
  • 내부 IP 주소
    • 변경할 내부 IP 주소를 입력합니다.
  • 서브넷 마스크
    • 변경할 내부 네트워크의 서브넷 마스크를 입력합니다.


  • 허브/AP모드 내부 게이트웨이
    • 공유기를 허브/AP 모드로 사용 시, 인터넷에 연결하기 위해 내부 게이트웨이와 내부 DNS 주소를 설정합니다.
