• DHCP 서버 설정
    • DHCP 서버는 PC에 IP 주소를 자동할당해 주는 역할을 담당합니다.
  • DHCP 서버 동작
    • 현재 공유기의 DHCP 서버를 실행/중지 할 수 있습니다.
  • IP 주소 대여 범위
    • DHCP 서버가 PC에 자동할당해 줄 수 있는 IP 주소의 범위를 설정 할 수 있습니다.
    • 예를 들어, '192.168.0.[10] ~ 192.168.0.[100]'으로 설정할 경우에는 PC들은 192.168.0.10 부터 IP 주소를 자동할당 받게 됩니다.
  • 서브넷 마스크
    • DHCP 서버의 서브넷 마스크를 설정할 수 있습니다.
  • 게이트웨이 주소
    • DHCP 서버의 게이트웨이 주소를 설정할 수 있습니다.
  • 기본/보조 DNS 주소
    • DHCP 서버의 DNS 주소를 설정할 수 있습니다.
  • DNS suffix
    • DNS suffix를 정의하면 자동으로 클라이언트 이름뒤에 도메인 이름이 붙게 됩니다.
    • 예를들어, DNS suffix를 'office.com'으로 지정한 후, 'abc'라는 클라이언트 이름으로 ping 테스트를 하게되면 "abc.office.com" 도메인 이름에 대한 DNS 쿼리가 송신됩니다.


  • DHCP 서버 주소관리
    • 특정 PC의 MAC 주소와 IP 주소를 지정하시면, 내부적으로 고정 IP주소를 사용하는 것과 동일하게 사용하실 수 있습니다.
    • "검색된 IP/MAC 주소"의 '추가' 버튼을 누르면 선택되어 있는 IP/MAC 주소를 지정할 수있습니다.
    • 검색된 않은 IP/MAC 주소를 등록할려면 첫번째 줄의 체크박스를 클릭한 후, IP/MAC 주소를 입력하면 됩니다.
    • "설정된 IP/MAC 주소"의 '삭제' 버튼을 누르면 지정된 IP/MAC 주소를 삭제할 수 있습니다.
