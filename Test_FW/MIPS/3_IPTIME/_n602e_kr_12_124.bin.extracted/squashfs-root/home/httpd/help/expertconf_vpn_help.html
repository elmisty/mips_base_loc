• VPN 서버 설정
    • PPTP 서버
      • PPTP VPN 서버의 동작 상태를 설정할 수 있습니다.
      • 암호화(MPPE) 사용 : PPTP VPN 서버의 암호화 사용 여부를 설정할 수 있습니다.
      • 동작 중일 때 다음의 패킷은 포트포워드 되지 않습니다 : TCP/1723, GRE

    • L2TP 서버(일부 제품에서 지원)
      • L2TP/IPsec VPN 서버의 동작 상태를 설정할 수 있습니다.
      • 비밀키 : PSK 인증 방식에 필요한 사전 공유키(Pre-Shared Key)입니다.
      • 동작 중일 때 다음의 패킷은 포트포워드 되지 않습니다 : UDP/500, UDP/1701, UDP/4500, ESP

  • VPN계정 설정
    • VPN 접속에 필요한 계정,암호,할당될 IP 주소를 설정 합니다.
    • 할당될 IP주소는 동적IP 주소범위와 겹치지 않게 합니다.
    • 연결 상태 : 해당 계정의 VPN연결 접속 상태
    • 연결 끊기 : VPN 연결을 강제 종료
    • 삭제 버튼 : 계정을 삭제
    • 같은 계정으로 다수의 연결을 할 수 없습니다.
    • 관리자 계정 및 암호를 설정하지 않고 VPN계정 추가 시, 보안코드를 입력해야 합니다.

  • 설정 예
    • VPN접속 계정 : iptime
      VPN접속 암호 : iptimeisbest
      IP 주소 : 192.168.0.241
